# COASTAL SPIRIT DEMO WEBSITE

This is a demonstration website created for educational purposes to show improved web design principles compared to the Weird Fish website.

## DESIGN IMPROVEMENTS DEMONSTRATED:

1. ABOVE-THE-FOLD CLARITY
   - Single clear hero section with one primary CTA
   - No intrusive cookie popups at page load

2. CLEAN NAVIGATION
   - Simple, sticky header with clear categories
   - Active state indicators

3. BRAND STORY PROMINENCE
   - "Our Story" section clearly visible
   - Accessible from main navigation

4. CONSISTENT DESIGN LANGUAGE
   - Coastal colour palette throughout
   - Friendly but professional tone

5. ACCESSIBILITY FEATURES
   - Skip link for keyboard users
   - Semantic HTML
   - Alt text placeholders
   - Good colour contrast

6. PERFORMANCE AWARENESS
   - Lazy loading on images
   - Preconnect for fonts
   - Minimal JavaScript

## TO USE:

1. Add your own images to the /images folder
2. Replace placeholder image references in index.html
3. Customise colours in :root variables in style.css
4. Test with Lighthouse for real performance scores

## FILE STRUCTURE:
- index.html - Main page structure
- css/style.css - All styling
- js/main.js - Simple interactions
- images/ - Add your images here
